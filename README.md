## 🌟 About The Project

**WanderLuxe** is a cutting-edge frontend web application designed for luxury travel and tour bookings. Built completely from scratch, it abandons basic templates to offer a bespoke, high-end user interface. With cinematic dark themes, smooth glassmorphism effects, and fluid animations, WanderLuxe delivers an online experience as breathtaking as the destinations it promotes.

### ✨ Key Features

- **Premium UI/UX:** A bespoke dark cinematic theme complemented by metallic gold and amber accents.
- **Glassmorphism Design:** Modern translucent UI elements layered dynamically over high-resolution imagery.
- **Flawless Responsiveness:** Engineered to scale perfectly across massive 4K displays, tablets, and mobile devices without losing its luxury feel.
- **Interactive Search:** A sleek, minimal booking form to filter by destination, date, and guests.
- **Trending Destinations:** Interactive grids showcasing exclusive locations like Bora Bora, Paris, Tokyo, and the Swiss Alps.
- **Optimized Performance:** Blazing fast load times utilizing Vite and zero heavy CSS frameworks.

## 🛠️ Technology Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** [Google Fonts](https://fonts.google.com/) (Outfit & Inter)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js installed on your machine.

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Karankumar2403/WanderLuxe.git
   ```

2. Navigate into the project directory
   *(Note: If your directory contains special characters like `&` on Windows, wrap the path in quotes or rename the folder.)*
   ```bash
   cd "WanderLuxe"
   ```

3. Install NPM packages
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`

## 🎨 Design System

The project relies entirely on a custom-built Vanilla CSS architecture (`src/index.css`), ensuring maximum control. 

- **Primary Colors:** Midnight Blue (`#080c14`, `#111827`)
- **Accent:** Metallic Gold (`#D4AF37`)
- **Typography:** `Outfit` (Headers) and `Inter` (Body text)
